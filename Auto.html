Autonmous Vehicle

Three UCI students and I had worked in a team to create a vehicle that will autonomously naviagate a track and reach the the furthest distance. However the task was not so simple as we were to design a propulsion system using a golf cart tire releasing air through a solenoid valve which was to push a piston powering a gear assembly which we had designed. All being controlled by an Arduino Uno which we fabricated code for to maximize the efficiency of the propulsion system to allow for the longest period of driving

Project Tasks and Milestones

Electrical Diagram: As the team member acting as the Electrical Engineer I was in charge of all the electrical components and the powering and testing of these components. Using an engineering software I was able to create the documentation for the components and how they were to be assembled.

Testing of Components: To test the components were operating, a team member and I formulated code on Arduino IDE to check for multiple components. First if the solenoid would release air at the correct interval. Then, to ensure the steering system would work we had tested the range of motion the servo would turn. We also were to make sure the switch was correctly stopping signal when pressed and passing signal when on. Lastly, we were to make sure once the magnetometer was calibreaed that it would display the correct orientation when it was turned.
